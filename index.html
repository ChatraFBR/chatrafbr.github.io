Lenguajes de marcas

SGML
Standard Generalized Markup Language (SGML); es decir: Lenguaje de Marcado Generalizado Estandar.Es un lenguaje basado en texto que se puede usar para marcar datos, es decir para añadir metadatos, que se autodescriben. Estas marcas permiten estructurar el texto de manera logica y tambien como debe presentarse en pantalla. SGML es un lenguaje que fue estandarizado mediante la Norma ISO 8879 en el año 1986.
XML
XML son las siglas del Lenguaje de Etiquetado Extensible. La expresion se forma a partir del acronimo de la expresion inglesa eXtensible Markup Language. Se trata tambien de un lenguaje estandar que posee una Recomendacion del World Wide Web Consortium
HTML
(Hipertext Markup Language) o lenguaje de marcas de hipertexto es el lenguaje que permite la generacion de hipertextos en la World Wide Web. El lenguaje HTML deriva de SGML pero, a su vez, ha tenido unos desarrollos posteriores
